Tables
Tag/Attribute Function
<table> Creates a new table
<thead> Sets the table headers
<tbody> Sets the main content of the table
<tfoot> Sets the table footer (can be written anywhere in <table>)
<tr> Creates row
<th> Creates a column with header text
<td> Creates cell data
colspan="2" Number of columns a cell covers
rowspan="2" Number of rows a cell covers

Iterators
Method Function
(i) => {**code**} Callback function. Iterators require one to work and typically use arrow function syntax.
.forEach() Executes callback function on each element of an array.
.map() Like .forEach(), but saves results to a new array; those matching condition have value of 'true'.
.filter() Similar to .map(), but filters results by condition and saves them to the new array.
.findIndex() Uses callback function to return first matching array element.
.reduce((a,c)=>{}, *initial) a = accumulator
c = currentValue
*initial = Optional 2nd argument adds value to result.
Takes a callback function with two parameters (a,c), adds elements and returns a total value.
.some() Tests if array has at least one element which passes test of callback function. Returns 'true' or 'false'.
.sort() Sorts array elements in ascending order according to Unicode. Returns results in new array.
Use compareFn to sort by:
(a,b) => a-b //ascending order
(a,b) => b-a //descending order
.every() Tests if all array elements pass test of callback function. Returns 'true' or 'false'.
